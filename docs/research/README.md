# Research Guide

`docs/research/` 现在只保留仍有背景价值的讨论记录与历史分析。

这些文档的价值在于解释：

- 为什么曾经这么想
- 某些策略或参考资料是怎么进入项目的
- 某些判断为什么后来被修正

它们不是当前实现真相。

## 使用原则

- 需要当前真相时，先读 `docs/overview/` 和 `docs/design/`
- 需要理解历史背景和推理过程时，再读这里
- 如果研究结论已经成为稳定架构约束，应提升到 `docs/design/` 或 `docs/overview/`
- 阶段实验报告与已失效的研究记录应移动到 `docs/archive/research/`

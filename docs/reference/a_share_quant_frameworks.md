# A股开源量化基础设施调研报告

这份文档是 A 股开源量化基础设施的背景调研，用于帮助评估底层研究与执行框架。

调研聚焦于 GitHub 上主流的开源量化框架，从 **A 股生态兼容性**、**回测/实盘性能**、以及与 **AI 大模型的契合度** 三个维度进行对比。

---

## 1. 微软 Qlib (Microsoft Qlib) —— AI 驱动的最佳底座
*建议评级：⭐⭐⭐⭐⭐ (极度推荐)*

Qlib 并不是一个传统的“回测引擎”，而是一个专门为了 **AI 机器学习量化投资** 而生的全栈式平台。

*   **设计理念**：将量化研究抽象为“预测 -> 组合优化 -> 订单执行”的标准化流水线 (Pipeline)。
*   **A股兼容性**：极佳。内置了对中国 A 股市场（如沪深 300, 中证 500）的高度优化，提供了一键下载 A 股日线数据的预处理脚本。提供了经典的 Alpha158 和 Alpha360 标准化因子数据集范式。
*   **对于 Pixiu 的价值**：
    *   **因子工厂**：LLM 只需要吐出符合 Qlib 语法的因子表达式语言 (Expression Alpha)，Qlib 的底层 C++ 引擎就会以极高的性能完成全市场计算。
    *   **机器学习无缝对接**：内置了 LightGBM, XGBoost, 甚至各种图神经网络 (GNN) 模型。我们可以让 LLM 生成因子，直接喂给内置模型打分，形成完美闭环。
*   **缺点**：学习曲线极其陡峭，架构宏大而复杂，部分高频实盘接口不如传统框架成熟。

## 2. VN.PY (VeighNa) —— 走向实盘的必经之路
*建议评级：⭐⭐⭐⭐ (实盘利器)*

VN.PY 是国内最著名的开源量化交易系统。它由拥有丰富实战经验的交易员开发，主打稳定性和多市场接入。

*   **设计理念**：事件驱动 (Event-Driven) 的实盘交易系统。
*   **A股兼容性**：完美兼容国内券商的柜台接口（如中泰 XTP, 华鑫奇点，恒生 UST ），同时也支持国内期货市场（CTP）。
*   **对于 Pixiu 的价值**：
    *   如果我们最终的目标是让 AI 产生的策略**不仅停留在回测，而是真实下单**，VN.PY 是不二之选。
    *   它提供了一个名为 `vnpy.alpha` 的模块，专门为了与机器学习策略进行对接。
*   **缺点**：底层更偏向实盘执行，其自带的回测引擎在进行全市场“截面多因子”组合回测时，性能和便利度不如专门的科研框架（如 Qlib 或 VectorBT）。

## 3. Backtrader & Zipline —— 经典的欧美派系
*建议评级：⭐⭐ (可作为备选，需二次开发)*

这两者是全球知名度最高的 Python 纯量化回测库。

*   **Zipline**：量化回测的祖师爷（曾经的 Quantopian 核心）。
    *   **痛点**：由于它以美股逻辑为核心，默认不支持 A 股的 T+1 规则、涨跌停限制，引入它需要做大量的“魔改”（虽然存在诸如 `zipline-chinese` 的开源项目，但维护状态不佳）。
*   **Backtrader**：极其灵活、连小学生都能很快上手写出策略的框架。
    *   **痛点**：对于单只股票（如 BTC 或者贵州茅台）的择时策略很好用，但如果我们让大模型同时分析 A 股 5000 只股票做**横截面选股**，Backtrader 纯 Python 循环的底层性能会是一场灾难。

## 4. Hikyuu (基于 C++/Python) —— 性能狂魔
*建议评级：⭐⭐⭐ (极客之选)*

这是一个相对没那么大众，但在性能上极其追求极致的国内开源项目。

*   **设计理念**：C++ 底层，Python 接口。类似于早期的通达信/大智慧逻辑，通过语言化公式来构建策略。
*   **对于 Pixiu 的价值**：如果在沙箱里进行高强度的“遗传算法”变异并回测上万个策略，Hikyuu 的并发速度会显著节省硬件资源。
*   **缺点**：生态不如前两者繁荣，社区支持和现成的周边工具较少，对于通过 LLM 生成其特定语法的代码存在较高的 Prompt Engineering 挑战。

---

## 5. 数据源基建建议 (Data Infrastructure)

沙箱再好，也需要真实的数据“喂养”。A 股数据获取是另一个大坑。

1.  **AKShare**：目前国内最大、最全、且更新极快的开源财经数据接口库。它是免费的，支持获取 A 股全市场历史数据、甚至包括财务报表、宏观经济数据、以及微博/雪球的舆情数据（非常适合我们的 NLP 情绪策略）。**核心推荐**。
2.  **Tushare Pro / BaoStock**：提供极度标准化的 Pandas DataFrame 格式数据。Tushare 的接口设计非常干净，特别适合 LLM 根据 API 文档自动编写数据获取代码。（注意：Tushare 高级权限需要积分，BaoStock 是完全免费离线的）。

---

## 6. 实用结论

结合 A 股环境、当前 Pixiu 的研究对象形态和确定性执行路径，**“Qlib + AKShare”** 仍然是最务实的基础设施组合：

1.  **分工明确**：AI Agent 不需要懂复杂的订单簿如何撮合，它只需要懂得如何像宽客一样去写 Alpha 因子公式。将公式交给 **Qlib**，Qlib 会自动处理去极值、标准化、正交化等所有脏活累活。
2.  **安全性高**：AI 生成的“因子代码”相比于生成任意 Python 交易脚本，其破坏力更小，更容易在沙箱中进行安全执行与校验。

这份结论应作为长期背景参考，而不是当前迭代计划。

# Pixiu v2 商业化架构原则

> 版本：2.0
> 创建：2026-03-09
> 前置依赖：`v2_architecture_overview.md`
> 状态：**Exploratory**

---

## 1. 目的

本文件只记录会影响技术架构的商业化原则，不涉及最终定价和上市节奏。

---

## 2. 资产边界

因子库和反思库从第一天起物理独立于引擎代码，存放私有仓库。

```
公开侧（可 fork）：
  src/                               → 引擎代码
  knowledge/skills/constraints/      → 公共约束知识
  docs/specs/                        → 架构规格

私有侧（永不公开）：
  data/factor_pool_db/               → 因子库
  data/reflections/                  → 反思库
  knowledge/skills/discovered/       → AI 发现的永久规范
```

---

## 3. 数据隐私承诺

用户的交易数据、持仓、股票池不上传。

- 用户侧：本地运行、价格数据、信号计算
- 服务侧：License 验证、因子库分发、必要的授权检查

---

## 4. 因子库分发

候选方案：

```
因子公式编译为字节码并打包为镜像层
运行时向 License Server 请求 session_key
session_key 只在内存中解密，不落磁盘明文
订阅到期后停止新版因子库发放
```

---

## 5. 目标市场

优先面向：

- 个人私募
- 独立量化交易员

选择原因：

- 有一定技术能力
- 决策链短
- 对本地部署和数据隐私更敏感

---

## 6. License 方向

- 引擎：BUSL-1.1（候选）
- 因子库 / 反思库：专有授权

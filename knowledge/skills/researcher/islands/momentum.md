# 动量因子 — A 股特异性指南

> Island-specific skill。仅在 Researcher 分配到 momentum island 时注入。

---

## 1. A 股动量与美股的核心差异

### T+1 交易制度
- A 股买入后次日才能卖出，短期动量信号的持有期至少为 T+1
- 日内动量（intraday momentum）在 A 股不可直接交易，只能作为信号而非策略
- 建议最短回看窗口：**5 日**（美股可以做 1 日）

### 涨跌停制度
- 主板 ±10%，科创板/创业板 ±20%
- 连续涨停时，Ref-based 动量因子会被截断
- 涨停日成交量极低，动量信号失真
- 建议：计算动量时排除涨跌停日，或使用对数收益率

### 散户主导效应
- A 股散户占比高，追涨杀跌行为放大短期动量
- 短期动量（5-20 日）通常强于长期动量（60-120 日）
- 美股常用的 12 个月动量在 A 股效果不稳定

---

## 2. 有效时间窗口

| 窗口 | A 股表现 | 适用场景 |
|------|----------|----------|
| 5 日 | 强，但换手率高 | 短线交易、事件驱动 |
| 10 日 | 较强，IC 稳定 | 通用短期动量 |
| 20 日 | 稳定，IC ≈ 0.03-0.05 | 主力推荐窗口 |
| 60 日 | 中等，regime 依赖 | 趋势跟踪 |
| 120-250 日 | 弱且不稳定 | 不推荐单独使用 |

---

## 3. 推荐的动量因子构建方向

### 价格动量
```
$close / Ref($close, 20) - 1
```

### 换手率调整动量（过度交易削弱真实趋势）
```
($close / Ref($close, 20) - 1) / (Mean($volume / Mean($volume, 60), 20) + 1e-8)
```

### 波动率调整动量（标准化后可比性更强）
```
($close / Ref($close, 20) - 1) / (Std($close / Ref($close, 1) - 1, 20) + 1e-8)
```

### 路径依赖动量（中间回撤影响持续性）
```
($close / Ref($close, 20) - 1) - Max(($close - Min($low, 20)) / (Min($low, 20) + 1e-8), 20)
```

---

## 4. 注意事项

- 动量因子在**趋势行情**中最有效，**震荡行情**中容易失效
- 建议配合 regime 检测使用 `applicable_regimes: [trending_up, trending_down]`
- 北向资金流入期间，动量因子 IC 通常提升
- 财报季（4 月、10 月）动量信号受事件冲击，IC 不稳定

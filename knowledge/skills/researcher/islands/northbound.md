## 2. 北向风格代理信号解读

### 趋势性增仓（强信号）
- 连续 5 日以上净买入
- 持股市值占流通市值比例持续上升
- 通常领先 A 股走势 1-3 周

### 短期波动（弱信号）
- 单日大幅净买入/卖出但次日反转
- 节前/月末的仓位调整
- 被动基金（ETF）调仓带来的噪声

### 与 A 股情绪的领先/滞后关系
- 北向资金通常**领先** A 股情绪 1-2 周
- 北向资金流入 + A 股下跌 = 潜在底部信号
- 北向资金流出 + A 股上涨 = 潜在顶部信号

## 3. 因子构建方向（代理）

由于 qlib 标准字段中不包含北向资金数据，以下为**代理因子**思路：

### 机构行为代理
```
# 大单净买入代理：高成交额 + 低换手率 = 机构增仓
($volume * $close) / (Mean($volume * $close, 20) + 1e-8) * (-1 * $volume / Mean($volume, 20))
```

### 外资偏好风格代理
```
# 北向偏好：大市值 + 低波动 + 高股息 的交集
# 使用价格和成交量代理
Mean($close * $factor, 20) * (1 / (Std($close / Ref($close, 1) - 1, 20) + 1e-8))
```

### 资金流入动量
```
# 成交额变化趋势（放量趋势 = 资金流入）
Mean($volume * $close, 5) / (Mean($volume * $close, 20) + 1e-8) - 1
```

## 4. Regime 依赖

| 市场状态 | 北向因子效果 |
|----------|-------------|
| 人民币升值期 | 强（汇率收益叠加） |
| 人民币贬值期 | 弱（资金回流压力） |
| 全球 risk-on | 强（新兴市场配置增加） |
| 全球 risk-off | 弱（避险资金撤出） |
| A 股与港股联动强 | 强 |

---

## 5. 注意事项

- 北向资金不等于"聪明钱"，其中包含大量被动 ETF 调仓
- 单日异常大额净买入可能是 MSCI / FTSE 指数调整所致
- 数据陷阱：沪深港通休市日不同步，需对齐日历
- 建议使用 5 日或 10 日移动平均来平滑噪声

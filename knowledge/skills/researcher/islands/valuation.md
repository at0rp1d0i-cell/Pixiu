# 估值因子 — A 股特异性指南

> Island-specific skill。仅在 Researcher 分配到 valuation island 时注入。

---

## 1. A 股估值环境的特殊性

### 散户定价 vs 机构定价
- A 股散户交易占比约 60-70%，小市值股票受散户情绪影响更大
- 机构定价的大盘股（沪深300 成分）估值因子更有效
- 小盘股估值经常偏离基本面（壳价值、题材炒作）

### 壳资源溢价
- 2020 年注册制改革前，壳资源溢价严重扭曲小市值估值
- 注册制后壳价值显著下降，估值因子在小盘股中开始变得有效
- 回测时需注意区分注册制前后的 regime

### 退市新规
- 2020 年后退市标准收紧，"面值退市"增多
- 低价股 ≠ 低估值，需区分"便宜"和"垃圾"
- 建议：估值因子配合质量过滤使用

## 2. A 股常用估值指标

| 指标 | 优势 | 劣势 | 适用条件 |
|------|------|------|----------|
| PE（TTM） | 直观，数据可得 | 周期股失真，亏损股无意义 | 盈利稳定的消费、医药 |
| PB | 对银行保险有效 | 轻资产行业失真 | 金融、周期品 |
| PS | 亏损公司也可用 | 忽略利润率差异 | 成长股、亏损科技股 |
| EV/EBITDA | 跨行业可比性好 | 数据可得性差 | 有 EBITDA 数据时优先 |
| 股息率 | 现金流质量信号 | 高股息可能是陷阱 | 成熟期企业 |

## 3. 纯价量估值代理

当缺少基本面字段时，可构建价量代理：

### 市值代理（低价效应）
```
1 / ($close * $factor)
```

### 换手率反转（高换手 = 过度定价）
```
-1 * Mean($volume, 20) / Mean($volume, 60)
```

### 价格偏离度（偏离长期均值 = 相对贵或便宜）
```
-1 * ($close / Mean($close, 120) - 1)
```

## 4. Regime 依赖

| Regime | 估值因子表现 |
|--------|-------------|
| 牛市中后期 | 弱（市场不看估值） |
| 熊市中后期 | 强（资金回归价值） |
| 震荡市 | 中等偏强 |
| 政策宽松期 | 弱（流动性驱动） |
| 政策收紧期 | 强（估值杀） |

建议 `applicable_regimes: [range_bound, bear_late]`
建议 `invalid_regimes: [bull_late, liquidity_driven]`

---

## 5. 注意事项

- A 股估值因子的 IC 通常在 0.02-0.04 之间，不要期望太高
- 估值因子是典型的**慢因子**，持仓周期建议 ≥ 20 日
- 避免在行业内部做估值排名时混合周期股和成长股
- 不要假设 `$pb/$pe_ttm/$float_mv/$turnover_rate` 一定可用；是否可用以当前 prompt 注入的 runtime 可用字段列表为准

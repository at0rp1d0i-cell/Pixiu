# 量能因子 — A 股特异性指南

> Island-specific skill。仅在 Researcher 分配到 volume island 时注入。

---

## 1. A 股量价关系的特殊性

### 散户主导的换手率含义
- A 股日均换手率远高于美股（A 股 1-3% vs 美股 0.3-0.5%）
- 高换手率更多反映散户情绪而非信息交易
- 换手率 = 成交量 / 流通股本，比绝对成交量更有意义

### 量价关系的 A 股特征
- **放量上涨**：趋势确认，但 A 股中后期放量可能是散户跟风见顶
- **缩量下跌**：抛压减弱，但 A 股中可能是无人接盘
- **放量滞涨**：主力出货信号，在 A 股特别有效
- **地量地价**：底部特征，但需要持续观察

## 2. 量能因子构建方向

### 换手率（标准化成交量）
```
$volume / Mean($volume, 60)
```

### 量比（即时量能相对历史）
```
Mean($volume, 5) / (Mean($volume, 20) + 1e-8)
```

### 量价背离（趋势持续性信号）
```
Corr(Rank($volume, 10), Rank($close, 10), 20)
```

### 异常成交量（突变检测）
```
($volume - Mean($volume, 20)) / (Std($volume, 20) + 1e-8)
```

### 成交额集中度（大单代理）
```
# 使用 vwap 偏离来代理大单行为
Abs($vwap / $close - 1)
```

### Amihud 非流动性
```
Mean(Abs($close / Ref($close, 1) - 1) / ($volume * $close + 1e-8), 20)
```

## 3. 量能分布的时间特征

### 开盘集中成交
- A 股开盘 15 分钟成交量占全天 10-15%
- 集合竞价反映隔夜信息消化
- 日频数据无法捕捉，但可通过 `$open vs $close` 间接推断

### 尾盘集中成交
- 收盘前 30 分钟成交量占全天 15-20%
- 机构调仓多在尾盘完成
- 可通过 `$close vs $vwap` 差异来推断尾盘方向

### 推断公式
```
# 尾盘买入代理：收盘价高于 vwap 说明尾盘有买入
$close / ($vwap + 1e-8) - 1
```

---

## 4. 与其他因子的交互

| 组合 | 信号含义 |
|------|----------|
| 高换手 + 高动量 | 热门追涨，短期有效但反转风险大 |
| 低换手 + 高动量 | "静默上涨"，通常更持久 |
| 量能突增 + 波动率上升 | regime 切换信号 |
| 持续缩量 + 低波动 | 市场底部特征 |

---

## 5. 注意事项

- `$volume` 单位是股（不是手），计算换手率时注意
- 停牌日成交量为 0 或 NaN，需要排除
- 新股上市前 5 日不设涨跌停，成交量异常，建议排除
- 成交额 = `$volume * $vwap`（近似），比纯成交量更稳定

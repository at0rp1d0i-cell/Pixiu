# 波动率因子 — A 股特异性指南

> Island-specific skill。仅在 Researcher 分配到 volatility island 时注入。

---

## 1. A 股波动率的特殊性

### 涨跌停截断
- 主板 ±10%，科创板/创业板 ±20%
- 已实现波动率被制度性截断：真实波动可能更大
- 涨跌停日的收益率被"压缩"，导致波动率估计偏低
- 建议：使用**高低价波动率**（Parkinson / Garman-Klass）补充

### 日历效应
- **周一效应**：周一波动率通常高于其他交易日（周末信息累积）
- **月末效应**：机构调仓导致月末最后 3 日波动率偏高
- **季末效应**：基金排名压力导致季末波动率异常
- **两会/重大会议期间**：波动率可能被人为压制

### 波动率非对称性
- A 股下跌波动率通常大于上涨波动率
- 散户恐慌性抛售放大下行波动
- 建议区分上行波动率和下行波动率

---

## 2. 波动率因子构建方向

### 已实现波动率（基础版）
```
Std($close / Ref($close, 1) - 1, 20)
```

### 高低价波动率（Parkinson 近似）
```
Mean(($high - $low) / ($close + 1e-8), 20)
```

### 下行波动率（只计算负收益）
```
Std(If(Lt($close / Ref($close, 1) - 1, 0), $close / Ref($close, 1) - 1, 0), 20)
```

### 波动率期限结构（短期 vs 长期）
```
Std($close / Ref($close, 1) - 1, 5) / (Std($close / Ref($close, 1) - 1, 20) + 1e-8)
```

### 波动率变化率（波动率的动量）
```
Std($close / Ref($close, 1) - 1, 5) / (Ref(Std($close / Ref($close, 1) - 1, 5), 5) + 1e-8)
```

---

## 3. 低波动异象（Low Volatility Anomaly）

- A 股存在**低波动异象**：低波动率股票长期回报优于高波动率股票
- 但效果有 regime 依赖：
  - 牛市中低波动策略跑输（错过高弹性标的）
  - 熊市中低波动策略显著跑赢
- 建议在 applicable_regimes 中标注

---

## 4. 与其他因子的交互

| 组合 | 效果 |
|------|------|
| 低波动 + 高质量 | A 股最稳定的 alpha 来源之一 |
| 波动率 + 动量 | 波动率调整动量通常优于原始动量 |
| 波动率 + 成交量 | 高波动 + 低成交量 = 流动性风险溢价 |
| 波动率期限结构 + regime | 短期波动率飙升是 regime 切换信号 |

---

## 5. 注意事项

- 波动率因子更新频率建议：**每日重算**（不适合月频更新）
- 涨跌停日的收益率需要特殊处理，建议检查 `$high == $low` 来识别
- 波动率因子在高频调仓中效果更好（5-10 日），月频调仓 IC 偏低

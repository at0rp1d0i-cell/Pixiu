# 情绪因子 — A 股特异性指南

> Island-specific skill。仅在 Researcher 分配到 sentiment island 时注入。

## 1. A 股情绪代理变量

### 可从价量数据构建的情绪代理
- **涨跌家数比**：全市场上涨股票数 / 下跌股票数（截面计算）
- **换手率异常**：相对历史均值的偏离度
- **涨停板数量**：当日涨停数 / 总股票数
- **量价配合度**：成交量与价格变动的一致性

### 外部情绪数据（当前 qlib 标准字段不含）
- 融资余额（需额外数据源）
- ETF 申赎比
- 期权隐含波动率（如 50ETF 期权）
- 投资者新开户数

## 2. 纯价量情绪因子构建

### 恐慌指标（下跌加速度）
```
# 近期跌幅加速 = 恐慌情绪上升
($close / Ref($close, 5) - 1) - ($close / Ref($close, 20) - 1) * (5/20)
```

### 散户情绪代理（换手率偏离）
```
# 换手率远超均值 = 散户情绪亢奋
$volume / (Mean($volume, 60) + 1e-8) - 1
```

### 量价背离（情绪反转信号）
```
# 放量不涨 = 情绪见顶
Corr($volume, $close, 10)
```

### 极端收益频率（尾部风险情绪）
```
# 近20日中收益率超过2%的天数占比（代理情绪亢奋度）
Mean(If(Gt(Abs($close / Ref($close, 1) - 1), 0.02), 1, 0), 20)
```

### 价格偏离长期均线（过度乐观/悲观）
```
($close - Mean($close, 60)) / (Std($close, 60) + 1e-8)
```

## 3. 情绪反转信号

### 时间尺度
- A 股情绪反转通常在 **3-5 个交易日** 内发生
- 极端情绪（恐慌/疯狂）的反转更快
- 温和情绪偏离的反转更慢（10-20 日）

### 反转特征
| 情绪状态 | 信号特征 | 预期反转 |
|----------|----------|----------|
| 极度恐慌 | 连续 3 日跌幅 > 5% + 成交量萎缩 | 反弹 |
| 极度亢奋 | 连续涨停 + 换手率 > 3 倍均值 | 回调 |
| 恐慌后平静 | 波动率从高位快速下降 | 企稳上行 |
| 亢奋后放量 | 创新高但成交量创更新高 | 见顶 |

## 4. 与政策事件的交互

### 政策驱动的情绪
- 央行降准/降息 → 短期正面情绪脉冲（1-3 日）
- 监管收紧（如限制融资、打击投机）→ 恐慌性下跌（3-5 日）
- 两会/中央经济工作会议 → 情绪波动压缩（会前）→ 释放（会后）

### 注意
- 政策事件的情绪影响有"钝化"效应：同类政策反复出台后效果减弱
- 重大政策的第一次冲击最大，后续边际递减
- 建议在 applicable_regimes 中标注 policy_driven

## 5. Regime 依赖

| Regime | 情绪因子表现 |
|--------|-------------|
| 高波动市 | 强（情绪信号最清晰） |
| 低波动震荡 | 弱（情绪平淡无信号） |
| 趋势上行 | 中等（顺势情绪有信号） |
| 急跌反弹 | 极强（恐慌反转最有效） |

---

## 6. 注意事项

- 情绪因子通常是**反转因子**，不是趋势因子
- 极端值比中间值更有信息量，建议非线性处理（如取分位数）
- 情绪因子 IC 不稳定，建议做 ensemble 而非单因子使用
- 避免将换手率直接等同于情绪——机构大额交易也会抬高换手率
